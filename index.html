Supports Qt's Event Loop.

AppImage Updater Bridge fully supports Qt Projects by flowing with its event loop and thus you will have no problem integrating AppImage Updater Bridge with your Qt Project.

Get Update Information Directly from your AppImage!

Read Update Information Directly from your AppImage with a very clean API , without any OS Specifics. Thus giving the best results on extracting Update Information from any AppImage.

Very Light Wieght!

This library was not only designed to be powerfull but also light weight than the official AppImage Updater library , As this library only depends on Qt5 , as long as you use Qt5, This library will never seem like a weight to you.

Clean C++ API.

This project is rated by Codacy with a A Project Certification and thus integrating this library will not affect your source , So this project is also best suited for those who care about code taste.

Why Create AppImageUpdaterBridge ?

AppImage's are cool but when it comes to updating package that uses AppImage , it can be pretty hard to workout. And Therefore I started creating a lot of libraries that can be used to AutoUpdate any software including AppImages , But AppImages used a seperate efficient way to update packages , Thus I started working on creating a nice bridge to this mechanism and So AppImage Updater Bridge was born.
